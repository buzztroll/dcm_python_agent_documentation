Uninstalling the AgentÂ¶

Perform the following 4 steps below with root authority to uninstall the python agent:

  1. Stop the agent
service dcm-agent stop
  1. Remove the dcm-agent package

Debian, Ubuntu Linux distributions

apt-get remove dcm-agent -y

CentOS, Fedora, RHEL distributions

yum remove dcm-agent -y
  1. Remove the symlinks for the service scripts used to start and stop the agent
find /etc/rc?.d -name "[SK][0-9][0-9]dcm-agent" -exec rm -f {} \;
  1. Remove the dcm user account
userdel dcm
