Filesystem
----------

By default the python agent is installed in the **/dcm** directory.  Subdirectories include:

* **bin** - where the agent code and bash scripts reside
* **cfg** - where user customization configuration files can be stored
* **etc** - where the agent's configuration files are stored including the main configuration file **agent.conf**
* **home** - the home directory for the Linux user (e.g. **dcm**) that the agent runs as
* **logs** - where the agent logs are stored
